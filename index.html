Thai Tutor

Thai Tutor has been my primary web project during the past month. Noticing the absence of a free and modern learning platform for the Thai language (Duolingo and LiveMocha do not currently provide support), I began the application with the idea that I could improve my coding while providing something truly useful to the world at the same time. In addition to basic HTML/CSS/Javascript I implemented the ReactJS framework for its practical component based approach and Bootstrap for its flexibility and ease of use. With future growth in mind I leveraged React to make my code object-oriented and reusable and used Git to track my work and control revisions. As of January 20, 2017 I am proud to say that Thai Tutor provides value to its users with the following accomplishments:

  • Teaches Thai with audio, writing, and also English based phonetics
  • Provides a powerful options menu which allows the student to focus their learning on their desired subject and format
  • Auto generates Thai numbers up to 10000 without need of any external API (fast and inexpensive for remote locations)
  • Responsive design gives both mobile and desktop users an attractive UI

Currently Thai Tutor runs as a solely client side single page application loading curriculum from a local JSON file but I am currently looking into creating a back end to provide the following:

  • User profiles and logins
  • Database containing user statistics
  • Potentially a connection to Google’s text to speech API, which is currently not officially supported. Google’s supported Speech Synthesis API does not currently offer Thai.
